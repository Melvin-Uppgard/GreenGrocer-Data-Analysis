# GreenGrocer-Data-Analysis
Business analytics project analyzing GreenGrocer sales, customers, and operations using Python. Includes descriptive, diagnostic, and predictive analysis with visualizations and insights.
# GreenGrocer Data Analysis

## Overview
This project analyzes a fictional grocery chain, GreenGrocer, using Python. It applies core business analytics techniques to evaluate performance, identify trends, and generate actionable business insights.

The project follows a full analytics workflow:
- Descriptive analytics
- Diagnostic analytics
- Predictive analytics
- Business recommendations

## Objectives
- Analyze sales and profit performance
- Understand customer segments and behavior
- Evaluate store efficiency and operations
- Identify trends and seasonal patterns
- Forecast future sales
- Provide data-driven recommendations

## Technologies Used
- Python
- pandas
- numpy
- matplotlib
- scipy
- scikit-learn

## Dataset
The dataset is generated within the script and simulates:
- Daily sales across multiple stores
- Department and category performance
- Customer demographics and spending behavior
- Store operational metrics

## Key Features

### Descriptive Analytics
- Total sales, profit, and average profit margin
- Sales by store and department
- Customer segment analysis
- Visualizations of sales distribution and trends

### Diagnostic Analytics
- Correlation analysis between store factors and performance
- Store efficiency comparison (sales per staff, per square foot)
- Seasonal analysis by month and day of week

### Predictive Analytics
- Linear regression model to predict store sales
- Department-level sales trend forecasting
- Model evaluation using R²

### Business Insights
- Top-performing store and department combinations
- Identification of underperforming areas
- Opportunity scoring by store

### Recommendations
- Increase investment in high-performing stores
- Expand high-margin departments
- Optimize staffing efficiency
- Target high-value customer segments
- Improve underperforming areas

## How to Run

1. Install required libraries:
pip install pandas numpy matplotlib scipy scikit-learn

2. Run the script:
python main.py

3. The program will:
- Generate sample data
- Perform analysis
- Display visualizations
- Output an executive summary

## Output
- Sales and performance metrics
- Data visualizations (bar charts, line graphs, scatter plots)
- Predictive model results
- Executive summary with insights and recommendations
